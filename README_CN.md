## 关于项目

Alibbit AI 客户端是一款功能强大的跨平台 AI 大模型客户端，将多个 AI 服务整合到统一界面中，让您在所有设备上都能流畅使用 AI。

## 核心特性

- **支持多种 AI 大模型**
  - ChatGPT (OpenAI)
  - Claude (Anthropic)
  - Gemini (Google)
  - Ollama (本地模型)
  - DeepSeek
  - 持续更新支持更多模型...

- **真正的全平台支持**
  - [macOS](../../releases)
  - [Windows](../../releases)
  - [Linux](../../releases)
  - [Android](../../releases)
  - [iOS](https://apps.apple.com/us/app/alibbit/id6754900796)

- **高级功能**
  - **MCP 工具集成** - 模型上下文协议支持，增强 AI 能力
  - **Artifact 预览** - 实时预览和交互生成的内容
  - 简洁直观的用户界面
  - 快速流畅的使用体验

## 快速开始

1. 下载适合您平台的最新版本
2. 安装并启动应用程序
3. 配置您的 AI 模型 API 密钥
4. 开始与 AI 对话！
