## About

Alibbit AI Client is a powerful, cross-platform AI model client that brings multiple AI services into one unified interface. Experience seamless AI interactions across all your devices.

## Key Features

- **Multiple AI Models Support**
  - ChatGPT (OpenAI)
  - Claude (Anthropic)
  - Gemini (Google)
  - Ollama (Local Models)
  - DeepSeek
  - And more...

- **True Cross-Platform**
  - [macOS](../../releases)
  - [Windows](../../releases)
  - [Linux](../../releases)
  - [Android](../../releases)
  - [iOS](https://apps.apple.com/us/app/alibbit/id6754900796)

- **Advanced Features**
  - **MCP Tools Integration** - Model Context Protocol support for enhanced capabilities
  - **Artifact Preview** - Real-time preview and interaction with generated content
  - Clean and intuitive user interface
  - Fast and responsive performance

## Getting Started

1. Download the latest release for your platform
2. Install and launch the application
3. Configure your preferred AI model API keys
4. Start chatting with AI!
